PROYECTO REDES 2025

Implementación Exitosa de Servidor Web en Entorno LAN Dinámico.

Detalles de la Defensa Técnica

Este proyecto se enfoca en la implementación y verificación de un servicio **Apache HTTP Server** en una LAN mínima, demostrando la capacidad de la laptop de funcionar como servidor y cliente.

Servicio (Capa 7)

Activación de **Apache/XAMPP**. Acceso verificado vía Loopback (http://webITI).

Red (Capa 3)

Uso de **DHCP Dinámico** (ej. 10.20.48.142). Confirmada la portabilidad entre distintas subredes.

Seguridad (Capa 4)

Configuración de **Firewall de Windows** para permitir tráfico **TCP/80**, asegurando el acceso remoto a clientes.

Punto de Acceso para Clientes

Para acceder al servidor desde cualquier máquina conectada a la misma red del salón de clases, use la siguiente dirección IP:

http://[Dirección IPv4 Dinámica]

*(Ejemplo: http://10.20.48.142)*
